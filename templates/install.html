Install SAVANE SPRL App

Follow these steps to install our app on your Android device

Open in Chrome

Open the SAVANE SPRL application in Chrome browser on your Android device.

Tap the Menu

Tap the three-dot menu icon in the top right corner of Chrome.

Select "Install App"

Select "Add to Home screen" or "Install App" from the menu.

Confirm Installation

Confirm the installation when prompted. The app will appear on your home screen.

If the button doesn't work, follow the manual steps above.

App Features
  • Works offline
  • Fast loading
  • Mobile-friendly interface
  • Secure data management
  • No app store required
  • Automatic updates
Troubleshooting

If you don't see the install option:

  • Make sure you're using Chrome browser
  • Check that you have a stable internet connection
  • Try visiting the app again after some time
